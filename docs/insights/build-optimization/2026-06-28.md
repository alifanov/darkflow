# Build Optimization — 2026-06-28

**Scope:** `docker-compose.yml`, `webapp/Dockerfile`, `webapp/package.json` (pnpm), `webapp/next.config.ts`, `webapp/.npmrc`, `webapp/docker-entrypoint.sh`, `.github/workflows/` (none present).

## Pipeline overview

Нет CI/CD — `.github/workflows/` отсутствует, релизы идут через `git push` на `main` (install one-liner тянет raw-файлы). Единственный реальный билд — Docker-образ webapp (multi-stage: `base → deps → builder → runner`, `node:24-alpine`, `pnpm install --frozen-lockfile`, standalone-output Next.js). Деплой webapp в проде — хостовый процесс `pnpm build && PORT=5555 pnpm start`; Docker-вариант webapp прячется за профилем `docker`. Postgres — `postgres:16-alpine` в Docker. Основное время теряется на передаче контекста сборки: `COPY . .` без `.dockerignore` шлёт **1.1 ГБ** демону каждый билд.
